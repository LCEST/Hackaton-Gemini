# Hackaton-Gemini

## Instalacion

Paso 1
- Clonar el repositorio 

Paso 2
- Instalar PNPM
    - npm install -g pnpm

Paso 3
- Instalar librerias
    - pnpm install

Paso 4
- Correr el proyecto
    - pnpm run dev
